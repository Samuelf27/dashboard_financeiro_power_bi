## 📌 Sobre o projeto

Projeto de **Business Intelligence** para análise financeira: importação da base em Excel, criação de medidas e construção de visualizações para acompanhamento de resultados.

## 📈 Indicadores

- 💰 Receita
- 💸 Despesas
- 🧾 Impostos
- 📈 Lucro

## 🛠️ Tecnologias

Power BI · Power Query · DAX · Excel

## 📂 Arquivos

- `Aula 4 - Do Zero.pbix` / `Aula 4 - Gabarito.pbix` — dashboard
- `Base Financeiro.xlsx` — base de dados

## ▶️ Como abrir

Baixe o arquivo `.pbix` e abra no **[Power BI Desktop](https://powerbi.microsoft.com/desktop/)** (gratuito).
